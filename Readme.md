# PostFlow

**PostFlow** — это современное мобильное приложение для просмотра контента и взаимодействия с публикациями. Проект представляет собой высокопроизводительный клиент с лентой постов и системой комментариев, работающей в реальном времени.

## 🚀 Стек технологий

* **Фреймворк:** [Expo](https://expo.dev/) (SDK 54)
* **Ядро:** [React 19](https://react.dev/) & [React Native 0.81](https://reactnative.dev/)
* **Навигация:** [React Navigation 7](https://reactnavigation.org/)
* **Стейт-менеджмент:** [MobX](https://mobx.js.org/) & [TanStack Query v5](https://tanstack.com/query/latest)
* **Типизация:** TypeScript
* **UI & Графика:** React Native SVG, Linear Gradient
* **Архитектура:** Включена поддержка New Architecture

## 🌐 Источник данных

Приложение опирается на открытое API для получения данных:
* **Документация API:** [https://k8s.mectest.ru/test-app/docs](https://k8s.mectest.ru/test-app/docs)

### Реализованный функционал:
* Загрузка ленты постов с пагинацией на основе курсора.
* Система комментариев с поддержкой обновления в реальном времени через **WebSocket**.
* Сортировка комментариев (новые/старые).
* Обработка состояний загрузки и ошибок (Error States).

## 🛠 Предварительные требования

Убедитесь, что у вас установлены:
* [Node.js](https://nodejs.org/) (LTS)
* [Expo Go](https://expo.dev/go) на смартфоне для быстрого тестирования.

## 📥 Установка и запуск

1.  **Клонирование репозитория:**
    ```bash
    git clone https://github.com/HasanovDoc/PostFlow-app
    cd PostFlow-app
    ```

2.  **Установка зависимостей:**
    ```bash
    npm install
    ```

3.  **Запуск сервера разработки:**
    ```bash
    npx expo start
    ```

### Команды запуска для платформ:
* **QR-код (универсальный):** Отсканируйте появившийся в терминале QR-код через приложение Expo Go на вашем смартфоне (iOS или Android) для мгновенного тестирования на реальном устройстве.
* **Android:** Нажмите `a` в терминале (требуется Android Studio / Emulator).
* **iOS:** Нажмите `i` в терминале (требуется Xcode / Simulator).
* **Web:** Нажмите `w` в терминале.

## ⚙️ Особенности конфигурации

* **Android Keyboard:** Используется режим `softwareKeyboardLayoutMode: resize` для корректного сжатия контента при вводе.
* **Edge-to-Edge:** На Android включен режим отображения контента под системными панелями для современного визуального стиля.
* **Safe Areas:** Использование `react-native-safe-area-context` для корректных отступов на устройствах с "челкой".
